Everything you need to manage agents

AgentRadar automatically detects background AI processes and keeps you informed so you can focus on other things while they work.

Auto-Detection

Scans running processes every 2 seconds for known agents like Claude Code, Gemini CLI, Aider, Codex, and more.

Live Status

Monitors CPU usage, memory consumption, uptime, working directories, and exact current activity per agent.

Smart Alerts

Native macOS notifications alert you immediately when an agent finishes its task or needs tool approval.

Deep Integration

Reads Claude Code JSONL transcripts for hyper-accurate status, showing exact thinking steps and tool commands.

Click to Focus

Click any agent row in the popover to instantly activate its specific terminal window and jump back into the action.

Privacy First

Everything runs locally. It never reads terminal output directly or network traffic. No data ever leaves your machine.
